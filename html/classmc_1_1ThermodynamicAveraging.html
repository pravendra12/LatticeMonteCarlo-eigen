double mc::ThermodynamicAveraging::GetThermodynamicAverage ( double  beta) const

Computes the thermodynamic average for the current sliding window.

This method calculates the weighted average of energy values using a Boltzmann-like distribution. It incorporates the inverse temperature (beta) and normalizes by the partition function.

Parameters
betaThe inverse temperature (1/kT) for weighting the average.
Returns
The computed thermodynamic average.
